# Nuxt-Admin-Template

一款Nuxt3管理模板，基于Vite、ElementPlus、Vue3和TypeScript。


## 使用

### 菜单meta定义

- [类型声明](app/index.d.ts)

- 修改meta定义后生成菜单

```
pnpm postinstall
```

## 问题

- Nuxt3 keepalive存在问题 [27401](https://github.com/nuxt/nuxt/issues/27401)，嵌套路由导致实现选择性缓存很难，所以全部使用扁平路由
